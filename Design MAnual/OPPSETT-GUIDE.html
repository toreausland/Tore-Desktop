Design Manual — Oppsett-guide for Claude.ai

Hva er dette?

Alt du trenger for konsistente dokumenter

Vi har laget en designmanual (en "skill"-fil) som forteller Claude nøyaktig hvordan alle Aurelian-dokumenter skal se ut. Når du laster denne inn i Claude, får du 100% konsistent styling på alle Word-dokumenter, presentasjoner, tabeller og diagrammer.

Du trenger kun én fil: aurelian-design-manual.md

Denne filen ligger allerede på skrivebordet ditt i mappen Design MAnual.

Metode 1: Claude.ai Projects

Anbefalt — fungerer for både chat og samarbeid
1
Gå til claude.ai
Åpne claude.ai i nettleseren og logg inn.
2
Opprett et Project
I venstre sidebar, klikk på "Projects" → "Create project". Gi det et navn, f.eks. "Aurelian Manufacturing".
3
Last opp designmanualen som Project Knowledge
I prosjektet, klikk "Add content" → "Upload files".
Finn filen aurelian-design-manual.md i mappen:
Skrivebord → Design MAnual → aurelian-design-manual.md
Last den opp.
4
Legg til Custom Instructions
Klikk "Set custom instructions" i prosjektet og lim inn teksten under:
Du er designansvarlig for Aurelian Manufacturing AS. VIKTIG: Følg ALLTID designmanualen (aurelian-design-manual.md) som er lastet opp som prosjektfil. Alle dokumenter, presentasjoner, tabeller, diagrammer og annet visuelt innhold skal følge denne manualen uten unntak. Spesielt: - Bruk KUN fargene definert i manualen (#F50537 rød, #2B2B2B svart, #6B6B6B grå) - Bruk KUN Calibri som font (Arial kun i diagrammer) - Presentasjoner: Hovedsakelig hvit bakgrunn med svart tekst. Svart bakgrunn kun på første og siste slide. Rød tekst for skjønnsmessige detaljer. - Word-dokumenter: Følg heading-hierarkiet, tabellstilene og komponentmønstrene i manualen. - Skrivestil: Profesjonell, presis, skandinavisk minimalisme. Data først.

Marker all tekst i den svarte boksen over, kopier (Ctrl+C) og lim inn (Ctrl+V) i Custom Instructions.

5
Ferdig!
Nå vil alle nye samtaler innenfor dette prosjektet automatisk følge designmanualen. Bare start en ny chat i prosjektet og be Claude lage et dokument.

Test det: Start en ny chat i prosjektet og skriv: "Lag en kort Word-rapport om produksjonskapasitet med en tabell og et KPI-panel" — og se at alt følger designmanualen automatisk.

Metode 2: Vanlig Claude-chat (uten Projects)

Enklere, men må gjøres hver gang

Hvis du ikke bruker Projects-funksjonen, kan du lime designmanualen direkte inn i chatten:

1
Åpne en ny chat på claude.ai
Gå til claude.ai og start en ny samtale.
2
Last opp filen som vedlegg
Klikk på binders-ikonet (📎) i chat-feltet → "Upload file".
Velg aurelian-design-manual.md fra:
Skrivebord → Design MAnual
3
Skriv instruksen sammen med filen
I samme melding som du laster opp filen, skriv:
Her er designmanualen vår for Aurelian Manufacturing. Følg denne 100% i alt du lager for oss i denne samtalen. Bruk den som referanse for farger, fonter, layout, tabeller og alt visuelt. [Din oppgave her, f.eks.: "Lag en presentasjon om produksjonskapasitet"]

Ulempe: Du må laste opp filen på nytt for hver ny samtale. Metode 1 (Projects) er bedre fordi den husker filen permanent.

Metode 3: Claude Code (CLI)

For deg som bruker Claude Code i terminalen (det du bruker nå)

Dette er allerede satt opp! Designmanualen ligger i prosjektmappen og Claude Code har tilgang til den automatisk. Du trenger ikke gjøre noe ekstra her.

Allerede klart: Når du jobber i Claude Code i denne mappen, kan du referere til designmanualen direkte.
