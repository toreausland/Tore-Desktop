Aker Solutions-konto — risikofaktorer

RisikoSannsynlighetKonsekvensMitigering
Eksisterende leverandørrelasjoner Høy Høy Posisjonér som tilleggskapasitet. Aker Solutions' ordrebok vokser raskere enn eksisterende leverandørbase. Led med kapasitetsargumentet.
Manglende API 6A/17D-erfaring Medium Høy Start med ISO 9001 og NORSOK som basis. Bygg API-kompetanse gjennom ettermarkedsleveranser. Vurder tredjeparts rådgivning.
Geografisk avstand til Ågotnæs Lav Medium Våler–Tranby er 1.5 timer (akseptabelt). Ågotnæs (Bergen) er lengre, men primærinnkjøp sitter på Fornebu/Tranby. Logistikk løses med transportavtale.
Lang kvalifiseringstid (subsea) Høy Medium Bruk Subsea Services (ettermarked) som brøhode. Kortere kvalifisering gir tidlige ordrer mens hovedkvalifisering pågår parallelt.
Fabrikken forsinket Lav Høy Pre-kvalifisering er papirbasert og kan starte før fabrikken er operativ. Dialog og relasjonsbygging starter nå, produksjon fra aug 2027.
Oljeprisfall påvirker ordrebok Lav–medium Høy Aker Solutions' diversifisering mot renewables og CCS gir motstandskraft. Ettermarked (Services) er mindre konjunkturfølsomt enn nyproduksjon.
