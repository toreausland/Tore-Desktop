Subsea Services & Aftermarket — FMCs raskeste vei til Aurelian

Subsea Services dekker alt vedlikehold, reparasjon og rekvalifisering av installert subsea-utstyr gjennom dets levetid (typisk 20–30 år). Dette inkluderer erstatningsdeler for ventiler, aktuatorer, bonnetter og konnektorer — samt spesialverktøy for installasjon og intervensjon. Aftermarket-segmentet er strategisk viktig fordi det gir lavere kvalifiseringsterskel, raskere ordresyklus og repeterbare bestillinger.

Tjenester og produkter

  • Reservedeler til installerte trees — ventilgater, seter, bonnetter, aktuatorstempler, pakningsrør
  • Rekvalifisering av returnert utstyr — maskinering av nye tetningsflater, utboring av slitte boringer, re-maskinering til ny spesifikasjon
  • Spesialverktøy for intervensjon — running tools, pulling tools, test fixtures for nedsenkbart utstyr
  • Installasjonsverktøy — guide frames, alignment tools, connector running tools — prosjektspesifikke
  • Upgrade-kits — komponentoppgraderinger for eksisterende subsea-systemer til ny spesifikasjon

Aurelians posisjon

Hvorfor aftermarket er den raskeste veien inn hos FMC

Aftermarket-deler har lavere kvalifiseringsterskel enn nye tree-komponenter. Reservedeler og verktøy krever ikke full API 6A-kvalifisering for alle kategorier — mange kan leveres under FMCs generelle leverandørgodkjenning med materialssertifikater og dimensjonskontroll. Dette gir Aurelian muligheten til å få første ordrer 6–12 måneder før full tree-kvalifisering.

Nøkkelargumenter

  • Rask kvalifisering: Mange aftermarket-kategorier krever kun ISO 9001 + materialssertifikater + dimensjonsrapport — ikke full API-audit. Aurelian kan starte her.
  • HMLV er vare kjernevirksomhet: 1–20 enheter per ordre i varierende konfigurasjoner er nøyaktig profilen Aurelians autonome celler er designet for.
  • Kort leveringstid: Aftermarket-deler har ofte hasteordrer — nedetid på subsea-utstyr koster operatøren millioner per dag. Aurelians 24/7 drift gir kortere responstid.
  • Bygger track record: Suksessfulle aftermarket-leveranser etablerer Aurelian i FMCs leverandørsystem og bygger referanser for høyverdi-segmenter.
  • Materialoverlapp: Inconel 718, Super Duplex, Stellite — identiske materialer som i nye tree-komponenter. Kompetansen overføres direkte.

Tilnærmingsstrategi

Fem steg for å entre Subsea Services

  1. Start med test fixtures og verktøy — lavest kvalifiseringsterskel. Aluminium og verktøystål, 3–4 akse, kort leveringstid. Bygger relasjon uten tung sertifiseringsprosess.
  2. Tilby «fast-track» reservedelskapasitet — pitch: «24/7 autonom drift = kortere responstid på hasteordrer enn tradisjonelle verksteder.» Subsea-nedetid koster millioner per dag.
  3. Demonstrer Inconel 718 kompetanse — ventilgater og -seter er høyverdi aftermarket-deler. Vis prosesspaner og overflateruhet-dokumentasjon.
  4. Forhandle blanket purchase order — åpen rammeavtale for standardreservedeler med call-off. Reduserer FMCs administrative belastning.
  5. Bruk aftermarket-referanser til å åpne tree-segmentet — etter 3–6 måneders suksessfulle leveranser, pitch overgang til nye tree-subkomponenter.

Progresjonsstrategi

  • Fase 1 (Q3 2027): Test fixtures, adapter, enkle verktøy — aluminium og karbonstål. Bygger leverandørprofil i FMC-systemet.
  • Fase 2 (Q4 2027): Aktuatorstempler, -sylindre og enkle reservedeler i Super Duplex/316L. Øker materialkompleksiteten.
  • Fase 3 (Q1 2028): Bonnet-deler og ventilkomponenter i Inconel. Demonstrerer presisjon på kritiske tetningsflater.
  • Fase 4 (Q2–Q3 2028): Overgang til nye tree-subkomponenter basert på aftermarket-track record. Full API 6A-kvalifisering.

Strategisk vurdering: Denne trinnvise tilnærmingen minimerer FMCs risiko og Aurelians kvalifiseringskostnad. Hver fase bygger kompetanse og kredibilitet for neste. Innen 18 måneder fra første aftermarket-ordre kan Aurelian være kvalifisert for tree-komponenter.
