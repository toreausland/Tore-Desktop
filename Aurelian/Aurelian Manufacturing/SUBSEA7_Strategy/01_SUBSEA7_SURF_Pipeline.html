SURF & Pipeline — Subsea 7s kjernevirksomhet

SURF (Subsea, Umbilicals, Risers, Flowlines) er Subsea 7s største og mest CNC-intensive forretningssegment. Segmentet leverer komplette subsea-installasjoner inkludert rørledningslegging, strukturmontasje og tilkoblingsarbeider på havbunnen.

Kjerneaktiviteter

  • Pipeline End Terminations (PLETs) — kritiske endepunkter for rørledninger som krever presisjonsmaskinererte konnektorhus og tømmeventilkomponenter
  • Pipeline End Manifolds (PLEMs) — fordelingsstrukturer på havbunnen med komplekse maskinerte tilkoblingspunkter
  • In-line strukturer — t-stykker, ventilstasjoner og strømningskontrollmoduler som krever nøyaktig maskinering
  • Konnektorsystemer — mekaniske og hydrauliske konnektorer som kobler subsea-infrastruktur sammen
  • Strukturelle klemmer og støtter — rørklemmer, beskyttelsesstrukturer og festeelementer for havbunnsinstallasjoner

Aurelians posisjon

Hvorfor Aurelian er relevant for SURF & Pipeline

SURF-segmentet er prosjektdrevet med varierende komponentbehov fra installasjon til installasjon. Tradisjonelle maskinverksteder som er optimalisert for serieproduksjon sliter med denne profilen. Aurelians autonome CNC-celler er designet for nøyaktig denne typen fleksibel, prosjektbasert HMLV-produksjon.

Nøkkelargumenter

  • HMLV-profil matcher perfekt: Hver SURF-installasjon krever unike komponenter tilpasset prosjektets spesifikasjoner. Aurelians cellekonsept håndterer hyppige omstillinger uten produktivitetstap.
  • 5-akse kapabilitet for komplekse konnektorer: Hub-assemblyer og konnektorhus krever 5-akse simultan maskinering med stramme toleranser. Dette er Aurelians kjernekapabilitet.
  • Materialekspertise i korrosjonsbestandige legeringer: Duplex, super-duplex og inconel er standard subsea-materialer. Aurelians maskinpark kan konfigureres spesifikt for disse materialene.
  • Digital sporbarhet for DNV-krav: Subsea-komponenter krever full materialsporbarhet og prosesskontroll iht. DNV-regler. Aurelians digitale tråd leverer dette som standard.
  • Kort leveringstid på prosjektspesifikke deler: 24/7 autonom drift betyr at Aurelian kan levere raskere enn tradisjonelle verksteder på urgente prosjektbehov.
  • Skalerbar kapasitet: Subsea 7s prosjektportefolje varierer — Aurelian kan skalere opp og ned uten faste kapasitetsforpliktelser.

Materialfokus

Subsea-materialer — hva Aurelian må levere

Subsea-komponenter opererer i ekstremt korrosive miljøer på havbunnen. Materialvalg er kritisk og styres av DNV-regler og NORSOK-standarder. Aurelian må demonstrere kompetanse på følgende materialer:

MaterialeTypisk brukMaskineringsutfordringAurelian-relevans
Duplex rustfritt stål (UNS S31803) Strukturelle klemmer, rørstøtter, beskyttelsesstrukturer Høy hardhet, arbeidsherding Kjernemateriale — konfigurer maskinpark for dette
Super-duplex (UNS S32750) Konnektorhus, hub-assemblyer, ventilhus Svært høy hardhet, krever spesialverktøy Høy verdi per del — differensiator
Inconel 625 / 718 Cladding, tetningsflater, kritiske konnektordeler Ekstremt vanskelig, høy verktøyslitasje Mulighet for spesialisering
6Mo (UNS S31254) Sjøvannseksponerte komponenter Ligner super-duplex i vanskelighetsgrad Relevant for sekundære strukturer
Karbonstål (S355 / API 5L) Primærstrukturer, beskyttelsesrammer Standard — ingen spesielle utfordringer Grunnlastmateriale, høyt volum

Strategisk anbefaling: Aurelian bør konfigurere minimum én CNC-celle spesifikt for duplex/super-duplex maskinering. Dette inkluderer verktøy med PVD-belegg, høytrykks kjølevæske og prosessparametere optimalisert for arbeids­herdende materialer. Denne kapabiliteten er en differensiator i det norske subsea-markedet.
