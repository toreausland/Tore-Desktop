# Non-Conformance / Concession Records

| Field | Value |
|-------|-------|
| **Document No.** | AM-NCR-2026-0042-001, -002 |
| **Revision** | 0 |
| **MRB Section** | 20 |
| **SDRL Reference** | 8.3 |
| **SDRL Category** | 8 — Reports |
| **Format** | HTML (A4 Portrait) |
| **Branding** | Aurelian Manufacturing |

## Purpose

Documents any non-conformances identified during manufacturing and their resolution. Includes disposition (use-as-is, repair, rework, scrap) and customer concession where required.

## Content

- NCR identification and date raised
- Description of non-conformance
- Root cause analysis
- Proposed disposition
- Customer concession request/response (if applicable)
- Corrective action taken
- Verification of corrective action
- Closure sign-off

## Notes

- NCRs reported to customer within 24 hours per SDRL requirement
